# WBC 2026 Pool C Tiebreaker Simulator

Streamlit app that simulates the three-way tiebreaker scenario in Pool C of the 2026 World Baseball Classic.

If Korea beats Australia, Korea, Australia, and Chinese Taipei all finish 2-2. The WBC tiebreaker is decided by the lowest ratio of runs allowed to defensive outs across head-to-head games. This app shows who advances as the #2 seed for every possible Korea–Australia scoreline, including mercy rule scenarios.

## Run

```bash
pip install -r requirements.txt
streamlit run wbc-tiebreaker.py
```
